The Unusual SkyBlock 12.0.9 をDLしていただきありがとうございます

[インストール方法]
シングルの場合はMinecraftのセーブデータが格納されているフォルダに本マップを格納してください。
マルチの場合は、それぞれお使いのサーバーソフトにしたがって本マップを格納してください
またマルチサーバーの設定も必要で、server.propertiesをテキストエディタで開き、spawn-protection=0 と enable-command-block=true と allow-flight=true を追記してください。
すでに存在する場合は、上記の設定に書き換えてください。
また、eula.txtを開き、eula=falseの場合はEULA規約に同意したのちeula=trueにしてください

[非公式サーバーソフトに関して]
非公式サーバーソフトは公式サーバーソフトとは異なった動作をすることがあります。
非公式サーバーソフトでの正常なプレイは保証できませんので、ご理解の上ご利用ください。

[MODに関して]
本マップは遊ぶためにMODの導入は必要ありません。
またユーザー様のほうでMODを導入して遊んでいただくのはかまいませんが、サポートのほうはできかねます。

[バージョンの読み方]
マインクラフト自体のアップデート(x).島の追加(y).バグフィックス(z)のx.y.zでの記載になります。

[二次配布や動画、生放送での使用に関して]
本マップは二次配布や改造して配布することが可能です。報告や配布元の記載も必要ありません。
動画や生放送での使用も同様に可能です。報告や配布元の記載は必要ありません。
